drvOmronEIPConfigPoller() - Creates a new poller with user provided settings and adds it to the driver.

Parameters
[in]portNameThe name of the asynPort connected to the omron driver which will create this poller.
[in]pollerNameThe name of this poller, this needs to be referenced by records that need to use this poller.
[in]updateRateThe time in seconds between polls.
[in]spreadRequestsRather than sending requests as fast as possible (default=0), we spread requests over the polling interval to create a smoother load on the plc and network.
